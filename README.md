# CricTactix AI 🏏

CricTactix AI is an advanced, AI-powered Cricket Tactical Analyst Platform. It leverages Retrieval-Augmented Generation (RAG) and structured data analytics to provide deep, contextual cricket insights, player comparisons, and matchup intelligence through a natural language interface.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** FastAPI
- **AI & RAG:** LlamaIndex, ChromaDB, OpenAI

*Currently in early development.*
